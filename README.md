# NovaFlow AI Workspace

This repository is split into two separate apps:

- `frontend/` - the Vite + React application
- `backend/` - the API scaffold

## Common commands

```bash
pnpm dev
pnpm build
pnpm lint
pnpm dev:backend
```

## Notes

- The current product UI lives in `frontend/`.
- The backend is intentionally lightweight for now so it can grow into the API stack you want next.
